# SITTA - Tugas Praktik 1 STSI4209

Website **SITTA (Sistem Informasi Tiras dan Transaksi Bahan Ajar)** Universitas Terbuka.

## 📂 Isi Folder

```
sitta-praktik/
├── index.html          → halaman login
├── dashboard.html      → menu utama, laporan, dan histori
├── tracking.html       → cek status pengiriman paket
├── stok.html           → lihat dan kelola stok bahan ajar
├── css/
│   └── style.css       → semua tampilan
├── js/
│   ├── data.js         → data dummy (user, buku, dll)
│   └── script.js       → semua logika tombol, form, tabel
└── assets/
    └── img/            → gambar cover bahan ajar
```

## 🚀 Cara Pakai

1. Ekstrak folder `sitta-praktik/`.
2. Buka file **`index.html`** di browser (Chrome, Firefox, atau Edge).
3. Login pakai akun contoh berikut:
